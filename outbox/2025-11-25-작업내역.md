## 7. 활동수수료 페이지 버튼 표시 문제 해결

### 문제 상황
- 초기화면 → 지원금 → 활동수수료: "내 한도 확인하기" 버튼 보임
- 초기화면 → 금융캠퍼스 → 하단 푸터 → 지원금 → 활동수수료: 버튼 안 보임
- 새로고침하면 다시 보임

### 원인 분석
- 금융캠퍼스 CSS의 `.detail-card { display: none; }` 스타일이 전역적으로 적용됨
- 활동수수료 페이지의 `.detail-card`에도 동일한 스타일이 적용되어 버튼이 숨겨짐
- Next.js 클라이언트 사이드 라우팅 시 CSS가 완전히 언로드되지 않아 충돌 발생

### 해결 방법
금융캠퍼스 CSS에 더 구체적인 선택자 적용

### 수정 파일
- `smartlink/app/info-gfe/info-gfe.css` (173-181번째 줄)

### 변경 사항
```css
/* 변경 전 (전역 적용) */
.detail-card {
  display: none;
}

.detail-card.active {
  display: block;
}

/* 변경 후 (금융캠퍼스 페이지에만 적용) */
.page.active .detail-card {
  display: none;
  margin-top: 1.5rem;
}

.page.active .detail-card.active {
  display: block;
}
```

### 효과
- 금융캠퍼스 페이지의 `.detail-card`만 영향을 받음
- 활동수수료 페이지의 `.detail-card`는 영향 없음
- 다른 페이지도 안전함
